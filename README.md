## Description

Faisal Daily Blog is a personal blog website that I developed for almost a year. At the beginning of development, Faisal Daily Blog experienced many problems, such as incorrect backend logic, website design that was too monotonous, and also features that were not really needed. However, now I have resolved this deficiency, by presenting the newest version of the Faisal Daily Blog. Now, Faisal Daily Blog is not only an ordinary personal blog website, but also a blog platform that has hundreds of article posts that are useful for readers out there.

## Usage

If you are curious, just visit https://www.faisaldailyblog.com/
